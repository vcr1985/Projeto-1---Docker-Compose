Bem-vindo ao Projeto Docker Compose

Este é um exemplo simples de um projeto usando Docker Compose.

Para começar, certifique-se de ter o Docker e o Docker Compose instalados em sua máquina.

Você pode iniciar os serviços definidos no arquivo docker-compose.yml com o seguinte comando:

docker-compose up

Para parar os serviços, use:

docker-compose down
